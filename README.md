SaaS management dashboard for enterprise users. Configure data sources, run analysis workflows, view insight reports.

## Quick Start

```bash
pip install -r requirements.txt
python3 main.py
# → http://localhost:8080
```

## Features

- **Data source marketplace** — configure and connect data sources
- **4-step configuration wizard** — set up analysis in minutes
- **Report viewer** — visualized insights with CSS charts
- **Pricing page** — free / ¥29 / ¥99 / ¥499 tiers
